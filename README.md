# Pixelfont

When you want a font, but are kinda lazy.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     pixelfont:
       github: sleepinginsomniac/pixelfont
   ```

2. Run `shards install`

## Usage

```crystal
require "pixelfont"
```
